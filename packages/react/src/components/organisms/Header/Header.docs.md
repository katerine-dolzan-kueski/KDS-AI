# Header

Header component description will be added here.

## Usage

```tsx
import { Header } from '@kueski/react';

function App() {
  return (
    <Header>
      Header content
    </Header>
  );
}
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `undefined` | Additional CSS classes |
| `children` | `ReactNode` | `undefined` | Header content |
