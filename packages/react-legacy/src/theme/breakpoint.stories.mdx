# Breakpoints

KDS’s layouts and grids use uniform elements and spacing to encourage consistency across platforms, environments, and screen sizes.

## Principles

### Mobile First

We focus on mobile first experiencies with consistent UI regions and spatial organization.

### Consistent

Layouts should use grids, keylines, and padding consistently.

### Responsive

Layouts are adaptive. They react to input from users, devices, and screen elements.
