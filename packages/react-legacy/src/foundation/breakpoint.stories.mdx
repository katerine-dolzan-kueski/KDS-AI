# Breakpoints

Layouts are adaptive. They react to input from users, devices, and screen elements.

## Responsive

When creating a responsive layout system, it's helpful to establish minimum and maximum dimensions
for the body and margins, as well as scaling behavior that allows each region to adapt to different
form factors. The following guidance describes KDS's baseline dimensions and behavior.
