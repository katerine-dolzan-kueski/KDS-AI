# Colors

A system of additional colors from the primary, secondary and semantic colors.

## Extended Color Palette

<Grid $cols="9" $gap="spacing05" $symetric>
  <DesignSystemGlobals />
  {Object.entries(color).flatMap(([name, colors]) =>
    !(colors instanceof Object) ? (
      <ColorCard color={colors} title={name} />
    ) : (
      Object.entries(colors).map(([variant, colorValue]) => (
        <ColorCard color={colorValue} title={`${name}-${variant}`} />
      ))
    ),
  )}
</Grid>
