# Font Stack

We use two different font stacks at Kueski.
The first one that is applied to Headers, Titles, Sub-titles, CTA's and actionable components,
and the second one that is applied to captions and body text.

## Font Family

<Card>
  <DesignSystemGlobals />
  <Grid $cols="2" $gap="spacing06" $symetric>
    <Text $format="Body/Large/Bold">Variant</Text>
    <Text $format="Body/Large/Bold">Value</Text>
    {Object.entries(font.family).map(([name, value]) => (
      <React.Fragment key={`family-${name}`}>
        <Text>{name}</Text>
        <Text>{value}</Text>
      </React.Fragment>
    ))}
  </Grid>
</Card>
